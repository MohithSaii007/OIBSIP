# 📊 Unemployment Analysis in India

## 📌 Project Overview

This project analyzes the **unemployment situation in India** using data science techniques.  
It explores unemployment rates across different areas and compares key economic indicators such as employment and labour participation rate.

The goal is to understand unemployment patterns and visualize insights using Python.

---

## 📂 Dataset Information

The dataset contains the following features:

- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

---

## 🎯 Objectives

- Analyze unemployment rate across different areas
- Compare employment and labour participation
- Identify patterns and trends in data
- Visualize insights using graphs

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Visualizations Performed

✔ Unemployment Rate by Area  
✔ Average Unemployment by Area  
✔ Employment vs Labour Participation Rate  
✔ Correlation Heatmap
