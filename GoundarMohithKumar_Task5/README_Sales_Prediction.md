# Sales Prediction using Machine Learning

##  Project Overview
This project is part of the Oasis Infobyte Internship. It predicts product sales based on advertising spending using Machine Learning.

##  Dataset Features
- TV: Advertising budget on TV
- Radio: Advertising budget on Radio
- Newspaper: Advertising budget on Newspaper
- Sales: Target variable

##  Objective
To predict sales based on advertisement spending.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn

##  Model
Linear Regression (Regression Problem)

## Workflow
Load data → Clean data → Train model → Predict → Evaluate

##  Evaluation
R² Score and Mean Absolute Error used.
